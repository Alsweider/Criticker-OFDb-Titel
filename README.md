# Criticker-OFDb-Titel
Userscript für Tampermonkey, das die deutschen Filmtitel von OFDb.de abruft und auf Criticker-Filmseiten als Untertitel anzeigt. 

Download: [Greasyfork](https://greasyfork.org/de/scripts/558803-criticker-ofdb-titel)
